# PYTHON-PROJECT-FOR-DATA-ANALYSIS--EXPLORATORY-DATA-ANALYSIS
Data Analyst Project

📊 Diwali Sales Analysis
📌 Project Overview

This project analyzes Diwali sales data to understand customer purchasing behavior, sales trends, and key insights that can help businesses improve their marketing strategies.

🎯 Objectives
Identify top-selling products during Diwali
Analyze customer demographics (age, gender, etc.)
Find which states/cities generate the most revenue
Understand purchasing patterns
🛠️ Tools & Technologies Used
Python 🐍
Pandas
NumPy
Matplotlib / Seaborn
Jupyter Notebook
📂 Dataset Information
Dataset contains customer details like:
User ID
Gender
Age
State
Product Category
Purchase Amount

(Agar dataset ka source hai to yahan likh dena)

📊 Key Analysis Performed
Data Cleaning (missing values, duplicates removal)
Exploratory Data Analysis (EDA)
Visualization of:
Sales by Gender
Sales by Age Group
Top States by Orders
Product Category-wise Sales
📈 Key Insights
Females contributed more to total sales
Age group 26–35 has highest purchasing power
Certain states generate maximum revenue
Electronics & Clothing are top-selling categories
